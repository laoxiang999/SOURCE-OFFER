# 国际贸易代理网站（GitHub Pages 版）
多语言外贸网站，支持中文、英文、俄文、乌克兰文切换，适配全球访问。

## 网站功能
- 响应式设计（支持电脑/手机/平板）
- 多语言切换（中/英/俄/乌）
- 产品服务展示
- 联系咨询入口

## 部署指南
1. Fork 本仓库到你的 GitHub 账户
2. 进入仓库 → Settings → Pages → 选择 "Deploy from a branch"
3. 分支选择 `gh-pages`，目录选择 `/ (root)`，点击 Save
4. 等待 1-5 分钟，访问 `https://laoxiang999.github.io/SOURCE-OFFER` 即可看到网站

## 自定义修改
- 替换 `images/` 文件夹中的图片（Logo、Banner、产品图）
- 修改 `lang.js` 中的文本内容（公司信息、联系方式等）
- 调整 `style.css` 中的样式（颜色、布局等）
